IAM Access Instructions

Follow these steps to grant read‑only access for the audit.

Step 1 – Create an IAM user
  1. Go to AWS Console → IAM → Users → Create user.
  2. User name: mrshpetny-audit
  3. ✅ Check "Provide user access to the AWS Management Console"
  4. ✅ Choose "Custom password" – set a strong password and copy it.
  5. ❌ Uncheck "User must create a new password at next sign‑in"
  6. Click Next.
Step 2 – Attach permissions
  1. Select "Attach policies directly".
  2. Search for ReadOnlyAccess → ✅ Select it.
  3. Click Next → Create user.
Step 3 – Retrieve Account ID
  1. After creation, go back to IAM → Users → click mrshpetny-audit.
  2. Copy the Account ID – the 12‑digit number in the top‑right corner of any AWS Console page (e.g., 123456789012).
